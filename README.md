# 🐄 Livestock Genetic Selection & Predictive Modeling — Balai Inseminasi Buatan Lembang

## 📌 Project Overview
Delivered a comprehensive computational selection and predictive system to optimize livestock breeding and nutritional management. The solution implements Python-based Genetic Algorithms for superior bull selection and Machine Learning models for feed ration automation, serving as a core technological upgrade for Balai Inseminasi Buatan (BIB) Lembang to meet national agricultural standards.

## 🏗️ Data Architecture, Pipeline & Impact

1. **⚙️ Database Engineering & Normalization:** Engineered and normalized the facility's livestock database to **3NF in MySQL**, successfully eliminating data redundancies and ensuring structural integrity for complex computational queries.
2. **🗃️ Genetic Algorithm Implementation:** Developed a robust bull selection system in Python utilizing Genetic Algorithms, carefully calibrated to adhere to strict **SNI (Standar Nasional Indonesia) KPIs**.
3. **📊 Predictive Modeling for Nutrition:** Deployed a Machine Learning model to dynamically automate feed ration formulas, specifically optimized to meet precise **Dry Matter (Bahan Kering)** nutritional requirements.
4. **🎯 Business Impact:** Successfully **improved overall fitness scores by 27.5% across 5 generations** through automated, data-driven genetic selection, while simultaneously streamlining the daily feed calculation process.

## 🛠️ Tech Stack Used
* **Programming & ML:** Python
* **Optimization Algorithms:** Genetic Algorithms (GA)
* **Database Management:** MySQL (Relational Modeling, 3NF Normalization)
* **Domain Expertise:** Livestock Genetics, Feed Ration Formulation, SNI KPI Calibration
